This directory contains the following files:

readme.txt		This file

changes.txt		Changelog of KTelnet, KFTP and KPopProxy.

default.map		Default keymap settings.

central.map		Translation tables between ISO 8859-2 and CP1250.
			This table should be merged into the profile
			when 'Centraleuropean' character encoding have
			been selected.

meta.map		Keymap to map Alt-A to Alt-Z into the character
			sequence ESC A to ESC Z. Nice to have together with
			emacs.

xterm-vt220.map		Some keymaps to select different sequences to
xterm-r6.map		functions keys etc.
xterm-xfree86.map
